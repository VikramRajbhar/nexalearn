## ✨ Features
- ⚔️ Real-time 1v1 Battles via Socket.io
- 🤖 AI-powered post-battle analysis (Gemini)
- 🧠 Adaptive difficulty skill graph
- 📊 Skill tracking across 14 CS topics
- 🏆 Global, weekly and topic leaderboards
- 🛡️ Guild system with realtime chat
- 👤 Profile with achievements and stats
- 🔐 Email + Google OAuth authentication

## 🛠️ Tech Stack
| Layer | Technology |
|---|---|
| Frontend | Next.js 14, TypeScript, Tailwind CSS |
| Realtime | Socket.io |
| Database | Supabase (PostgreSQL + Realtime) |
| Cache | Upstash Redis |
| Auth | Better Auth + Google OAuth |
| AI | Puter.js (Gemini Flash) |
| Deploy | Railway |

## 🏃 Run Locally
```bash
git clone https://github.com/YOUR_USERNAME/nexalearn
cd nexalearn
npm install
cp .env.example .env.local
# Fill .env.local with your values
npm run start
```
